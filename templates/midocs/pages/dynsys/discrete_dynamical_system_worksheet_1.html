A one-dimensional {% intlink discrete_dynamical_system_introduction %}discrete dynamical system{% endintlink %} has a single {% intlink state_variable_definition %}state variable{% endintlink %} that evolves in discrete time steps. If use the variable $x$ for this state variable and the variable $n$ to index the time steps, then we write the value of the state variable at time step $n$ as $x_n$.

Creating a discrete dynamical system takes four steps.

  1. Determine the state variable.
  2. Define the notation. You need to specify the notation for
    1. the state variable, and
    2. the variable for time.
  3. Specify the rule for going from one time step to the next.
  4. Specify the initial conditions.

Let's practice with three examples. Complete the above four steps for each of the following systems.

  1. The track for a railway is laid down at a rate of 10 meters per day. There is no track before the first day of construction.
  2. A colony of bacteria doubles in population every hour. The population initially contains $10^5$ bacteria.
  3. After an injection of penicillin, its concentration in your blood decreases by 20% every 5 minutes. Immediately after the injection, the penicillin concentration is 200 Î¼g/ml.

Once you have set up the dynamical system, the next step is to compute the future values. For each of the three systems you created, compute the value of the state variable for the next four time steps. (The length of each time step, of course, is different for the different problems. However, in your notation, you should still have numbered them 0, 1, 2, 3, etc.)

Once you have computed these values, sketch a graph of state variable versus time.
