The state space of a {% intlink dynamical_system_definition %}dynamical system{%endintlink%} is the set of all possible states of the system. Each coordinate is a {% intlink state_variable_definition%}state variable{%endintlink%}, and the values of all the state variables completely describes the state of the system. In other words, each point in the state space corresponds to a different state of the system.

An intuitive introduction to the state space is given in the {%intlink dynamical_system_idea%}idea of a dynamical system{%endintlink%}.
