A state variable is one of the variables used to describe the state of a {% intlink "dynamical_system_definition" %}dynamical system{%endintlink%}. Each state variable corresponds to one of the coordinates of the underlying {% intlink "state_space_definition" %}state space{%endintlink%}.

An intuitive introduction to state variables is given in the {% intlink "dynamical_system_idea" %}idea of a dynamical system{%endintlink%}.
