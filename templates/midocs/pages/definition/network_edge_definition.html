An edge (or {% intlink "network_link_definition" %}link{%endintlink%}) of a {% intlink "network_definition" %}network{%endintlink%} (or {% intlink "graph_definition" %}graph{%endintlink%}) is one of the connections between the {% intlink "network_node_definition" %}nodes{% endintlink%} (or {% intlink "graph_vertex_definition" %}vertices{%endintlink%}) of the network. Edges can be {% intlink "directed_graph_definition" %}directed{%endintlink%}, meaning they point from one node to the next, as illustrated by the arrows in the first figure below. Edges can also be {% intlink "undirected_graph_definition" %}undirected{%endintlink%}, in which case they are bidirectional, as illustrated by the lines in the second figure, below.
