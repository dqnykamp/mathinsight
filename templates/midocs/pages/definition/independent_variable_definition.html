An independent variable is a {% intlink "variable_definition" %}variable{%endintlink%} that represents a quantity that is being manipulated in an experiment. A {% intlink "dependent_variable_definition" %}dependent variable{%endintlink%} represents a quantity whose value depends on those manipulations.

In the context of a {% intlink "function_definition" %}function{%endintlink%}, the independent variables are the inputs to the function and the dependent variables are the outputs of the function.
